Intro to HTML

Fundamentals

  1. HTML is a markup langauge to give structure to text.
  2. CSS is for style.
  3. Javascript is for behaviour.
  4. An HTML element has a start tag <tagname>, an end tag </tagname> and some content inbetween. Some elements have self-closing tags (i.e. no end tag) and are called empty elements.
  5. Elements can also have attributes in the start tag to give additional instruction. Attributes often have name/value pairs, e.g. name="value".
  6. An HTML document has a head, which informs the browser on how to handle the page including the title and metadata (e.g. description, keywords) used by search engines, and a body, which contains the content of the page.

Tips

  1. It is good practice to indent lines of code to indicate hierarchy. The Beautify package in Atom auto-idents (Ctrl + Alt + B).
  2. The alternative text attribute (alt="") for the embed image element (<img src="">) is used by search engines to index pages, so ensure it is populated accurately.
  3. To open links in a new tab, use the attribute target="_blank" with the anchor element (<a target="_blank" href="">).
  4. There are many inputs to choose from when using forms. Forms need Javascript (behaviour) to become fully functional.
